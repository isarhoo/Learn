# Learn App (learn)

使用 Quasar 建立的示範專案，會透過 GitHub Actions 部署到 GitHub Pages。

## 安裝相依套件

```bash
yarn
# 或者
npm install
```

### 啟動開發模式（熱重載、錯誤回報等）

```bash
quasar dev
```

### 執行 Lint 檢查

```bash
yarn lint
# 或者
npm run lint
```

### 格式化程式碼

```bash
yarn format
# 或者
npm run format
```

### 建置正式版

```bash
quasar build
```

### 自訂設定

請參考 [Quasar 設定檔說明](https://v2.quasar.dev/quasar-cli-vite/quasar-config-js)。
